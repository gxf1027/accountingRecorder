日常生活流水账记录系统（学习系统 自娱自乐）

项目说明
▪ 日常记账：收入、支出、转账
▪ 明细显示
▪ 自定义查询
▪ 定时任务：月度/年度收入、支出统计，信用卡账单邮件发送，理财产品到期邮件提醒

技术框架：
▪ 核心框架：Spring framework 4.0+
▪ 安全框架：Spring Security 3.0+
▪ 视图框架：Struts2
▪ 持久层框架：MyBatis3
▪ 定时器：Quartz 2.3
▪ 数据库连接池：dbcp2
▪ 日志管理：log4j2
▪ 页面交互：jsp + jquery
▪ 会话管理：Spring-Session + Redis
▪ 缓存：ehcache
▪ 事务管理：Spring事务 + atomikos分布式事务
▪ RPC：weibocom/motan
▪ WebService：CXF

软件需求
▪ JDK1.8
▪ MySQL5.5+
▪ Tomcat8+
▪ ActiveMQ5+
▪ Redis
